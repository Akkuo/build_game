# 01. 系統與經濟拆解 (調用 Game Designer)

## 🎯 拆解目的
拆出競品「為什麼讓人一直想玩」的驅動力，以及「卡在哪裡逼人看廣告/付費」的商業化節點。

## 📋 執行步驟

1. **核心行為（0-30秒）拆解**
   - **輸入/輸出映射**：精確記錄操作模式（點擊、長按、滑動）。
   - **單次行動獎勵**：每次操作伴隨的數值或資源反饋為何？

2. **單局與留存鉤子（5-15分鐘）拆解**
   - **「再一局」誘因**：失敗時的接關條件（看廣告、假機率、差一步的心理暗示）。
   - **解鎖節奏**：第一個新機制在哪一分鐘出現？如何防止前期流失？

3. **逆向經濟模型表格**
   - **數值錄製**：以 Excel 記錄前 20 關（或前 60 分鐘）的玩家狀態、體力消耗、金幣掉落率與升級成本。
   - **動態難度調節 (DDA) 偵測**：測試連續失敗 3 次後，掉落物或過關門檻是否有暗中增益。

4. **廣告/內購 (IAP) 節點記錄**
   - **激勵/插屏/橫幅廣告**：記錄出現時機與倒數時間。
   - 記錄「破冰禮包」出現的心理卡點。

## 📦 對齊 Excel 規格書交付
完成拆解後，請將資料彙整成以下 Sheet 之格式：
- **`遊戲規則`**：詳細記錄輸入操作與通關條件。
- **`流程圖`**：核心循環機制與系統各介面間的跳轉關係。
- **`道具功能`** & **`機關物件`**：記錄競品中的加成道具與環境阻礙機制。
- **`製作項目`**：統整各單位所需完成的功能拆分。
