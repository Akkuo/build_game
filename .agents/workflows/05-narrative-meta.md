# 05. 主題與敘事包裝 (調用 Narrative Designer)

## 🎯 拆解目的
拆解休閒遊戲中的 Meta 故事與輕度包裝（如家裝、拯救、化妝解謎）如何緊繫玩家情感脈絡以推動長期留存。

## 📋 執行步驟

1. **主題與情感鉤子 (Hooks)**
   - 分析外部廣告買量素材與遊戲內實際主題的關聯性（例如：「背叛與復仇」、「拯救落魄母女」）。
   - 拆解這個故事引導如何無縫融入核心循環的「代幣產出/花費」機制。

2. **無文字環境敘事**
   - 場景如何只透過美術道具的切換來訴說狀態變化（例如：破舊的客廳 → 花費星星打掃 → 溫馨的房間）。

3. **角色情緒即時反饋**
   - 角色模型在畫面上方針對玩家操作做出的即時反應（如：連擊時歡呼、卡關時緊張流汗、過關的慶祝肢體語言）。

## 📦 對齊 Excel 規格書交付
完成拆解後，請將資料彙整成以下 Sheet 之格式：
- **`主題包裝`**：
  - 記錄競品是如何透過視覺題材包裝枯燥的益智玩法（例如：迷宮清場機制作為皮，農場動物作為肉）。
  - 記錄對應的輕度文字指引與世界觀背景設定。
