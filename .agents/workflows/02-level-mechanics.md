# 02. 關卡與空間分析 (調用 Level Designer)

## 🎯 拆解目的
拆解遊戲如何設計空間佈局、透過畫面引導視線，以及安排令人上癮的難度節奏。

## 📋 執行步驟

1. **關卡節奏譜分析**
   - 記錄關卡序列的節奏：如「教導（易）→ 應用（中）→ 爽快（極易）→ 卡關（極難，逼付費）」的排列規律。
   - 分析哪些關卡設計刻意製造心流破壞來引導變現。

2. **空間與防呆設計 (無文字引導)**
   - **熱區分析**：驗證 UI 與核心互動元素是否集中在螢幕下半部的「拇指舒適區」。
   - **視覺引導**：不看文字的情況下，遊戲透過哪些高亮、閃爍或動態效果直接引導玩家明白當下該點擊哪裡。

3. **破局工具配置**
   - 炸彈、洗牌等工具在關卡中出現的頻率與解鎖條件。
   - 分析這些工具如何影響盤面（如爆炸半徑、重新排序邏輯）。

## 📦 對齊 Markdown 工業標準產線 (Golden Layout)
完成拆解後，請將資料彙整成 Markdown 檔案，並且：
1. **[必備欄位]** 開頭必須標註 > 分析基礎： 與 > 負責人：。
2. **[視覺化]** 只要提到系統跳轉或狀態機，**強制使用 Mermaid Flowchart (graph TD)** 繪製流程圖。
3. **[硬核標籤]** 積極發掘以下技術指標，並用醒目的格式標示：
   - 物理手感偏移 (如 Y 軸防擋偏移 +150px)
   - 螢幕實體佔位 (強制 1080x2340 觀測盲點)
   - 效能與防發熱 (0 Overdraw 偽 3D、Object Pooling)
   - 動態音高多巴胺駭客 (Pitch Shifting)
   - 惡意情緒循環 (例如死局強制凍結)
