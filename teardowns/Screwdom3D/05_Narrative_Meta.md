# Screwdom 3D 規格書 - 05. 主題與敘事包裝
> 分析基礎：買量素材 (UA) 導引、皮相置換商業邏輯、挫折情緒剝削設計
> 負責人：Narrative Designer AI

## 1. 核心玩法的劇情抽離與皮相置換 (IP Dressing)

### 1.1 玩法與受眾的精準對焦
- **零文本載體**：Screwdom 3D 的核心玩法是純粹的拓撲排序邏輯解謎，無需也無法承載任何實質文本劇情。沒有角色對話、沒有世界觀說明，遊戲規則通過首關的「操作引導動畫」而非文字說明傳達。
- **皮相置換 (Reskinning) 的商業邏輯**：
  - **機械拆解主題**：工程師螺絲拆解感命中「精密、掌控、條理」的心理動機，吸引喜好智識優越感的男女性玩家。
  - **動物公仔主題**：小鴨子、兔子等可愛外形覆蓋在同一套螺絲邏輯上，立刻擴大受眾到喜好萌物的年輕女性玩家群體。
  - **節慶限定主題**：聖誕樹、南瓜燈等節日款，配合限時活動刺激非活躍玩家回流。

### 1.2 「破壞式成就感」的情緒核心
拆解是一種**被社會規範壓抑已久的本能衝動**。Screwdom 3D 精確鎖定了這個心理缺口：
- 把一個完整精密的 3D 模型，一顆螺絲一顆螺絲地系統性拆開，最後讓它完全崩解分離 —— 這種「越破壞越爽快」的情緒釋放，配合 ASMR 音效的加持，製造了極度獨特的心理滿足感。
- 這種成就感不需要任何的劇情敘事包裝，玩家透過視覺與聽覺本能地感受到「我做到了、我解開了」的自我效能感。

## 2. 買量敘事 (UA Narrative) 與情感勒索

### 2.1 挫折驅動與「失敗劇本」(Fail-State Drama)
Screwdom 3D 的買量廣告創意遵循以下公式（在業界被歸類為「Gameplay-First + Fail-State」模式）：
- **廣告場景 A（激怒型）**：一隻「假手」在螢幕上緩慢地、以明顯錯誤的順序取下螺絲，導致 Screw Jam 死局。觀看者天生的「你明顯應該先拆那顆！」智識優越感被瞬間激活。
- **廣告場景 B（懸念中斷型）**：廣告在玩家即將成功過關（最後 2 顆螺絲）的緊要關頭刻意切斷，用黑框文字打出「你能做到嗎？」，強迫觀看者帶著強烈的好奇心下載驗證。
- **廣告場景 C（對比型）**：前半段展示「錯誤的拆解困境」（卡死、死局），後半段展示「Expert Mode 的優雅連消」（流暢音階爬升、完美崩解），強烈對比刺激玩家想成為後者。

### 2.2 敘事心理學核心
- **智識優越感驅動**：「我一眼就看出哪顆順序是對的，那個廣告裡的人怎麼那麼笨」 → 這股情緒強迫玩家下載自我驗證。此策略在 CPI（每次獲客成本）上能壓出業界最低的折扣。
- **完成強迫症**：ASMR 金屬音配合模型逐漸被清空的視覺，製造驚人的「需要完成清空感」強迫心理，讓玩家在關卡快完成時極度不願意被廣告中斷。

## 3. 挫折與再挑戰體驗節點 (Failure Regret Points)

### 3.1 Screw Jam 懊悔凝視（2 秒情緒剝削）
當玩家的操作導致 Screw Jam 完全鎖死時：
- 系統在畫面上**高亮顯示所有卡死的螺絲路徑**（以紅色描邊標記），讓玩家痛苦地看清楚「是因為這顆被我拆早了才導致全盤皆輸」。
- 這種**強制的懊悔視覺化**讓玩家的大腦進入「下次我一定可以規劃得更好」的虛假自信迴路。
- 此刻彈出的 Extra Box 廣告（最強道具）點擊率達到全局最高峰，是精心設計的天選變現時機。

### 3.2 差一步的 Pity System 幻覺
- 系統 DDA 刻意讓約 70% 的卡關情境，都是「只要再多一個收集盒空間 / 只要再多一次 Undo」就能輕鬆解開的局面。
- 玩家感受到的不是「關卡太難了」，而是「我差一點就能自己解開了，是道具不夠才輸的」。
- 這種归咎外部因素（道具不足）而非自身能力不足的心理，大幅提升了廣告和 IAP 的轉換率。

## 4. Meta 系統的敘事黏著度 (Retention Hooks)

### 4.1 輕量模型解鎖 Meta（現階段）
- 以通關獲得的金幣或星星，解鎖收藏圖庫中的新 3D 模型主題（如解鎖「太空火箭組」、「古代神廟柱」等）。
- 目前的 Meta 深度較淺，屬於「給玩家一個繼續通關的目標」的圖鑑收集型設計，長線 D30 留存明顯弱於有 Home Renovation Meta 的競品（如 Magic Sort、Royal Match）。

### 4.2 未來 Meta 升級空間（策略預測）
若 Screwdom 3D 想進一步提升 D30 留存，需加入以下任一 Meta：
- **工坊建設 Meta**：用拆解累積的「零件」建造一座屬於玩家的「機械工作室」（類似 Magic Sort 的煉金塔）。
- **收藏館 Meta**：每拆完一個完整系列的模型，解鎖一個「展示架」可陳列已完成的模型縮圖，滿足收藏完整性強迫症。
