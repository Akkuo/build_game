# Magic Sort 規格書 - 05. 主題與敘事包裝
> 分析基礎：買量素材 (UA) 導引與 Meta 情感投射
> 負責人：Narrative Designer AI

## 1. 核心玩法的劇情抽離與皮相置換 (IP Dressing)
### 1.1 玩法與受眾的精準對焦
- **零文本載體**：如同所有的底層圖論解謎，Magic Sort 的主核心玩法 (Core Loop) 完全不需要、也無法承載實質的文本劇情。
- **皮相置換 (Reskinning for Demographics)**：為什麼要從 Water Sort 轉化為 Magic Sort (魔法分類)？這完全是出於獲取受眾量體的商業考量。
  - **視覺轉化**：普通的科學試管變成了帶有木塞的女巫燒瓶；彩色水疊加了底層的星光粒子變成了「魔法藥水」；死板的漸層背景被換成了「溫馨的煉金工房」。
  - **受眾紅利 (TA Expansion)**：這種美術與敘事包裝（皮相），成功將目標受眾從單一的「硬核數獨解謎玩家」，大幅外溢到「喜好休閒、神秘感、ASMR 體驗的 25-45 歲大眾女性玩家」。

## 2. 買量敘事 (UA Narrative) 與情感勒索
*敘事設計師在休閒遊戲中最大的影響力，不在於寫龐大的世界觀，而在於「編寫買量廣告腳本 (Ad Creatives)」。*

### 2.1 挫折驅動與「失敗劇本」 (Fail-State Drama)
- **經典廣告敘事公式**：畫面上半部是一個正在挨凍的女巫、或是被困在機關陷阱裡的可愛魔寵（強烈的情感弱勢方）；畫面下半部則是處於極其簡單狀態下的 3 管魔法藥水盤面。
- **激怒型操作 (The "Idiot" Player)**：廣告中會刻意展示一支假手，故意、緩慢地將明顯錯誤顏色的藥水倒在一起，導致死局。隨後，畫面上半部的陷阱因此觸發，導致弱勢角色遭到冰凍或岩漿吞噬。
- **敘事心理學**：這種敘事手法完全跳過了遊戲機制的說教，直接利用人類天生的「智商優越感」與「我都看得懂你怎麼那麼笨的強烈不甘心」。強烈的情緒波動迫使受眾產生親自下場破除荒謬劇情的衝動，進而點擊下載按鈕（這能將 CPI 獲客成本壓出驚人的折扣）。

## 3. Meta 系統的敘事黏著度 (Retention Hooks)
### 3.1 裝修與救贖系統 (Renovation Meta)
- 為解決純粹破關容易在中後期（LV200+ 以上）產生嚴重枯燥感與心理疲倦的危機，遊戲外圍必定會包裹一層極簡的「裝修改造 (Renovation) 敘事」。
- **敘事循環 (Narrative Loop)**：分類藥水推關（賺取魔法星星） -> 花費星星解鎖、打掃煉金塔的破舊家具（地毯、水晶球、掛畫）。
- **長期情感目標**：這套系統成功將玩家追求「枯燥數字通關」的單一物理維度，轉換為「把空盪殘破的房間整理得溫馨舒適」的情感維度。藉此把七日留存率甚至三十日留存率死死鎖住——此即所謂 Puzzle + Meta（混合休閒 Hybrid Casual）無可撼動的商業鐵律。
