# Magic Sort 規格書 - 01. 系統與經濟拆解
> 分析基礎：液體分類/水管分類 (Water Sort Puzzle) 核心機制
> 負責人：Game Designer AI

## 1. 核心玩法概要 (Gameplay Overview)
- **盤面基底**：多個裝有不同顏色液體的試管（或魔法藥水瓶），以及 1~2 個全空的備用試管。
- **基礎行為**：點擊「來源試管」使其浮起選中，接著點擊「目標試管」進行傾倒。每次傾倒會將來源試管最上層的連續同色液體，全部轉移至目標試管。
- **物理容積限制**：每個試管的標準容積通常為 4 單位液體（即最多 4 層）。
- **轉移規則**：液體**只能**倒入「顏色相同」的液體上方，或是倒入「完全淨空」的試管中。且目標試管必須有足夠的剩餘容積。
- **勝敗條件**：
  - **勝利**：所有瓶子要嘛是空的，要嘛裡面裝滿了單一顏色的液體（純色 4 單位）。
  - **失敗/死局**：畫面上既沒有空瓶，所有瓶子最上層的顏色也無法互相匹配倒入，且玩家用盡所有退回選項，即為「無法操作的死局」。

## 2. 遊戲核心與外圍流程 (Game Flow)
本流程圖對應至規格書中的 **`流程圖`** 分頁，明確定義單局生命週期與系統跳轉邏輯：

```mermaid
graph TD
    A[啟動遊戲 / 選擇關卡] --> B[進入藥水試管盤面]
    B --> C[點擊來源瓶 (瓶身微微浮起)]
    C --> D[點擊目標瓶]
    D --> E{是否符合傾倒規則?}
    E -- 否 (顏色不對或容積滿) --> F[瓶身輕晃並發出錯誤音效，退回原位]
    F --> C
    E -- 是 --> G[播放液體轉移與水流動畫]
    G --> H{是否有一瓶湊滿單一純色?}
    H -- 是 --> I[播放該瓶發光、封口蓋上特效與慶祝音效]
    I --> J{所有顏色皆分類完畢?}
    H -- 否 --> J
    J -- 否 --> K{是否陷入死局 (無步可走)?}
    K -- 否 --> C
    K -- 是 --> L[跳出提示：卡住了嗎？使用道具！]
    L --> M{看廣告解鎖 空瓶/重洗/退回?}
    M -- 是 --> N[獲得道具，脫離死局]
    N --> C
    M -- 否 (放棄) --> O[強制結算失敗 / 重新開始本關]
    O --> P[播放插屏廣告] --> A
    J -- 是 (勝利) --> Q[過關結算 / 金幣獎勵]
    Q --> R[播放插屏廣告] --> A
```

## 3. 操作邊界與極限反饋（0-30秒）
### 3.1 輸入/輸出映射與防呆 (Hitbox)
- **非線性二段點擊**：不採用「拖曳 (Drag & Drop)」而是採用「點擊 A 再點擊 B」的二段式操作，完全規避了手機螢幕邊緣拖拉容易斷觸的物理死穴。
- **取消選取防呆**：當點選 A 瓶浮起後，若改變心意，點擊畫面空白處或再次點擊 A 瓶，A 瓶會平滑降落取消選中，且不消耗任何負面數值。
- **極速連點測試 (Concurrency)**：傾倒液體的動畫通常約需 0.4~0.6 秒。系統**完全鎖定操作**，在動畫徹底完成前不接受下一次點擊。此舉避免了底層陣列資料與畫面的不一致。

### 3.2 單次行動反饋
- **視覺與聽覺**：傾倒時液體會有連續且平滑的出水與水位上升動畫，伴隨高還原度的「倒水白噪音 (ASMR)」。
- **自動結算 (Auto-completion)**：當整個盤面只剩下最後幾步必然的轉移動作時，部分系統會觸發 Auto-Resolve（自動完成跑完動畫），大幅降低後期的垃圾時間。

## 4. 單局目標層次與留存鉤子（5-15分鐘）
### 4.1 目標與難度生成
- **核心解謎依賴**：水管分類是典型的「排序演算法(Sorting)」圖論挑戰。玩家大腦必須佔用大量工作記憶來記住「如果我把綠色放到右邊，左邊的紅色就能出來」。
- **關卡生成公式 (RNG)**：這類遊戲沒有純隨機生成。所有關卡都是從一個「已完成的純色狀態」，由程式倒推隨機錯位打亂 N 步而產生的（確保該局擁有絕對解）。

### 4.2 死局與心流中斷
- 當玩家卡住時（步數死穴），遊戲不會立刻強制結束，而是**故意讓玩家在盤面上卡著乾瞪眼**。
- **強烈的挫折變現點**：此時畫面下方閃爍的道具（上一手 Undo、新增空瓶子）就成為救命稻草。這種從「極度挫折」到「看廣告瞬間破局」的落差，造就了驚人的廣告轉換率。

## 5. 逆向經濟與數值控制模型
### 5.1 核心經濟循環 (Sources & Sinks)
- **產出 (Sources)**：成功過關獲得金幣（看廣告可翻倍）。
- **消耗 (Sinks)**：使用金幣購買局內道具（撤銷、重洗），或是解鎖瓶身造型（Skin）、背景圖片。經濟系統極度封閉且缺乏通膨壓力。

### 5.2 道具體系與定價暗流
- **+1 空瓶 (Add Tube)**：這是水管分類遊戲破壞力最強的道具。加入一個空瓶等於增加了一個無限制的緩衝區 (Buffer)，難度瞬間歸零。
  - **定價策略**：這種毀滅級道具通常無法用遊戲內金幣購買，被嚴格限制為「必須觀看 30 秒激勵廣告」才能獲得，死死卡住最高 ARPU 的變現錨點。
- **撤銷/上一步 (Undo)**：容錯道具，通常每局提供 5 次使用額度，耗盡後須看廣告或花費高額金幣（削空玩家庫存）。

## 6. 廣告與 IAP 商業化節點
- **強插屏邏輯 (Interstitial)**：通關的成就感與挫折感交替之間（每過 1~2 關的結算畫面），強制彈出 15 秒不可跳過插屏廣告。
- **IAP 孤島**：除了 "$2.99 去除插屏廣告" 之外，主要 IAP 落在「無限次數 Undo + 限定版特效瓶子」的打包月卡訂閱 (Subscription) 設計上。
