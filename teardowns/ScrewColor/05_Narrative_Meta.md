# SCREWCOLOR 規格書 - 05. 主題與敘事包裝

## 1. 受眾定位與皮相邏輯
- **橋接型產品**：比 Screwdom 3D 門檻更低（2D 無空間旋轉），比 Magic Sort 視覺更新奇（金屬零件非液體），精準卡在兩大競品之間的受眾空白地帶。
- **皮相彈性**：核心算法不變，替換螺帽外形（六角帽 / 蝶形帽 / 方形帽）即可快速推出新主題版本，降低開發成本。

## 2. 情感鉤子
- **整理 OCD**：混亂螺帽被系統性分類完成的視覺，命中整理強迫症多巴胺迴路。
- **3 連消驚喜感**：消除動作是「系統自動觸發」而非「玩家預期的手動操作」，製造意外驚喜感，比手動消除的多巴胺釋放更強。
- **金屬 ASMR**：螺帽碰撞滑動的金屬質感音效，與 Screwdom 3D 的螺絲旋轉聲同屬金屬 ASMR，但側重「滑動碰撞」而非「機械旋轉」。

## 3. 買量廣告（UA Narrative）
- **場景 A（失敗型）**：廣告展示暫存槽被塞滿卻無一組可消除的窘境，觀看者產生「如果換我來，我不會這樣放」的智識優越感並下載。
- **場景 B（懸念型）**：廣告在最後一顆螺帽即將消除前硬切，附加「你能比他做得更好嗎？」引發未完成強迫症。

## 4. 挫折情緒設計（變現節點）
- **暫存槽視覺壓迫**：格子由藍 → 黃 → 紅的漸進警示，不需要任何文字就能讓玩家感受到「快不行了」的壓迫感，廣告轉換點精準落在「格子剩 1 個」的紅色警示時刻。
- **Pity System (70%)**：死局時幾乎永遠差一個格子；玩家歸咎道具不足而非關卡太難，IAP 轉換阻力最低。
- **2 秒懊悔凍結**：死局觸發後，全盤面暫停並高亮顯示問題螺帽 2 秒，讓玩家清楚看見「是哪步棋害死自己」，懊悔感最大化。

## 5. Meta 層現況與升級空間
- **現況**：輕量關卡通關制，部分版本以「解鎖新螺帽主題外觀」作為輕量 Meta。
- **D30 留存弱點**：缺乏強敘事 Meta，長線留存明顯遜於 Royal Match / Magic Sort 等有房間改造的競品。
- **建議升級方向（推測）**：工具箱收藏 Meta（用完成關卡累積的零件裝飾自己的工具間），可在不改核心玩法的前提下大幅提升 D30。
