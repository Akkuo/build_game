# SCREWCOLOR 規格書 - 03. 聽覺反饋擷取

## 1. BGM
- **風格**：輕快機械工廠感，鐵琴或馬林巴主旋律，BPM 約 105~115。
- **循環點**：第 8 / 16 拍換段，確保無縫 Loop。
- **情緒層次**：平常輕快 → 暫存槽快滿時張力弦樂加入 → 3 連消時短旋律爆發 → 過關完整勝利旋律。

## 2. 核心 SFX 清單

| 事件 | 音效描述 | Pitch Shifting | 備註 |
| :--- | :--- | :--- | :--- |
| 螺帽選中 | 輕金屬「叮」 | 無 | 短促清脆 |
| 移入暫存槽 | 金屬滑動「喀」 | 無 | ASMR 核心 |
| 3 連消觸發 | 清脆爆鳴 + 短旋律 | 每 Combo +0.5 音階 | 多巴胺核心時機 |
| 暫存槽警告 | 低頻警示「嗶」 | 無 | 配合紅框閃爍 |
| 死局鎖定 | 沉重「咚—」 | 無 | 全 UI 變暗同步 |
| 使用 +1 槽 | 清脆生成音 | 無 | 情緒逆轉正向音效 |
| 過關全清 | 4~8 拍勝利旋律 | 無 | 情緒釋放最高點 |

## 3. Pitch Shifting 設計（多巴胺駭客）
```
3 連消第 1 次：Do（基準）
3 連消第 2 次：Re（+0.5 半音）
3 連消第 3 次：Mi（+1.0 半音）
全色清空完成：高八度 Do（峰值獎勵音）
```

## 4. 技術規格
- **總預算**：≤ 12MB（2D 遊戲比 3D 少 30%）。
- **格式**：BGM AAC 128kbps；SFX OGG Vorbis Q6。
- **Audio Pooling**：最多 6 個聲源同時播放；超量 Cull 最舊聲源。
- **響度標準化**：-14 LUFS，確保各音效音量落差不影響 ASMR 體驗。
