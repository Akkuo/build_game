# SCREWCOLOR 規格書 - 04. 效能與美術拆解

## 1. 渲染策略（純 2D）
- **Shader**：Unlit（底色層）+ Alpha 混合（高光貼圖層），完全無即時光照計算。
- **螺帽金屬感實作**：預烘焙靜態高光貼圖（左上 45° 統一受光），所有顏色版本共用同一形狀 Mesh，以 Color Tint 區分顏色，節省貼圖資源。
- **Sprite Atlas 策略**：所有螺帽 Sprite + UI 元件合批入同一張 atlas.png，Draw Call 目標 ≤ 30 次 / 幀。

## 2. 素材規格
| 素材 | 類型 | 尺寸 | 備註 |
| :--- | :--- | :--- | :--- |
| 螺帽本體 | 2D Sprite | 128x128px @3x | 各色共用形狀，改 Color Tint |
| 螺帽高光層 | Alpha Sprite | 128x128px @3x | 固定白色高光，統一受光感 |
| 暫存槽底板 | 9-Slice | 192x192px @2x | 正常灰 / 警告橙 / 滿載紅 三態 |
| 收集盒 | 9-Slice | 256x256px @2x | 顏色對應螺帽，顯示收量數字 |
| 盤面背景 | 靜態圖 | 1080x2340px @1x | 工業木桌感，低飽和不搶螺帽 |

## 3. 動畫規格
| 動畫 | 觸發 | 實作 | 時長 |
| :--- | :--- | :--- | :--- |
| 螺帽選中浮起 | 點擊 | Scale 1.0→1.15 + 陰影放大 | 0.1s |
| 移入暫存槽 | 確認移動 | 拋物線 Lerp + 落定 Squash | 0.25s |
| 3 連消自動飛出 | 同色達 3 顆 | Bezier 飛入收集盒 | 0.3s |
| 暫存槽警示閃爍 | 剩餘 1 格 | Alpha 閃爍（0.5s 週期） | 循環 |
| 過關全清爆散 | 所有螺帽消除 | 星光粒子爆散 + 金幣飛出 | 1.0s |

## 4. 效能指標（目標值）
```
目標幀率：           60fps（全機型）
Draw Call 上限：     ≤ 30 次 / 幀
Overdraw 層數：      ≤ 2 層（2D 遊戲，比 3D 更嚴格）
貼圖記憶體預算：     ≤ 128MB
GC Allocation：      0（零動態記憶體分配）
啟動載入時間：       ≤ 2 秒
```

## 5. Object Pooling
- 盤面螺帽全部預初始化，關卡重置只改 Color Tint + Position，禁止中途 Instantiate / Destroy。
- 3 連消飛行粒子採 Object Pool（max 50 個），Alpha 在進入底層前強制淡出，Overdraw 嚴格 ≤ 2 層。
